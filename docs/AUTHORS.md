# Objetivo

Este documento identifica os colaboradores envolvidos no desenvolvimento do SGDE.

Cada colaborador possui um identificador único utilizado para rastreabilidade nos documentos do projeto.

---

# Convenção de Identificação

Os autores seguem o padrão:

```text
AUT-XXX
```

Onde:

- `AUT` representa Autor;
- `XXX` representa o número sequencial do colaborador.

Exemplo:

```text
AUT-001
```
